# react-website-1

A website built with React, using React Router, functional components, Hooks, a Navbar, Footer, and responsive design.

> Features
- Navigation between pages using React Router
- Functional components
- State management with useState
- Side effects handled with useEffect
- Fully responsive design
- Reusable Navbar components

> Create a <ins>***node_modules***</ins> directory in your current project directory if it doesn't already exist. :

npm install react <br />
